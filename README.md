# Телеграмм-бот погоды

Описание:
Проект представляет собой телеграмм-бота на Python, который предоставляет текущую информацию о погоде в четырёх городах: Москве, Сочи, Париже и Санкт-Петербурге. 
Бот использует API с сайта [OpenWeather](https://openweathermap.org/) для получения актуальных данных о погоде и библиотеку aiogram для создания и управления ботом.
